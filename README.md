## Projects

### PvP Engine
A real-time player-vs-player matchmaking and game state backend.

- Event-driven architecture using **Kafka** for decoupled game event processing
- **Redis ZSET** leaderboard with live rank updates
- **ELO rating system** for fair skill-based matchmaking
- Idempotent Kafka consumers to guarantee exactly-once game state transitions
- Containerized with **Docker Compose** for local development

### Android Biometric Auth
Intern project integrating face-based authentication into an Android application.

- Live face detection with **MediaPipe** and **CameraX**
- Performance optimization to reduce frame processing latency
- **TLS pinning** to protect biometric data in transit
